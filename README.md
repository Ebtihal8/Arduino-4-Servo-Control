## 🔌 Circuit Description

This project uses an **Arduino Uno** to control **four SG90 servo motors**. Each servo is connected to a dedicated digital pin, while power and ground are shared through the breadboard for synchronized operation.

### ⚙️ Hardware Components

- 🟦 Arduino Uno R3
- 🔄 4 × SG90 Servo Motors
- 📋 Breadboard
- 🔌 Jumper Wires

### ✅ Features

- 🤖 Controls four servo motors simultaneously
- ⚡ Uses the Arduino Servo library
- 🧪 Simulated in Tinkercad
- 📐 Simple and reliable circuit design
